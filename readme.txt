== Installation ==

1. Download the plugin, unpack it and upload the 'flickr-exif' folder to your wp-content/plugins directory.
2. To grant write access to wp-content/plugins/flickr-exif/cache. 
3. Activate the plugin through the 'Plugins' menu in WordPress.
